🍱 Meal Pass Management System

SAPL Industries Pvt Ltd — 150 staff + 2000 workers

1. Introduction

1.1 Purpose

This document defines the requirements for the Meal Pass Management System at SAPL Industries. The system manages employee meal access through barcode cards, tracks daily consumption, and integrates with attendance & payroll deduction. It ensures accuracy, speed, transparency, and real‑time reporting.

1.2 Scope

SAPL Industries has 150 Staff 2000 Workers The system will allow meal access using barcode‑enabled cards, identify employees, prevent duplicate meals, record history, calculate payroll deductions, and generate reports. Offline operation with periodic cloud sync is supported.

2. Overall Description

2.1 Product Perspective

Three major components: Admin Panel · Cafeteria Scan System · Cloud Reporting

Worker Card → Barcode Scanner → Local Scan System → Local Database → Cloud Sync → Admin Dashboard → Payroll Reports

2.2 Product Functions

  • ➕ Employee Management – add, update, assign barcode, activate/deactivate
  • 🍽️ Meal Tracking – record breakfast/lunch, prevent duplicate meals
  • ⚖️ Daily rules – workers: breakfast 1x, lunch 1x (total 2 meals/day)
  • 📊 Reporting – daily/monthly consumption, payroll deduction export

2.3 User Classes

👤 Admin (HR/management)

Manage employee database, configure meal costs, view reports, export payroll data.

🍴 Canteen Operator

Scan employee cards, serve meals, monitor system – no data modification.

🧑‍🏭 Employee (Worker/Staff)

Carry barcode card, scan during meal – no direct software interaction.

3. System Features

3.1 Employee Registration

Admin registers employees with: Employee ID, Name, Department, Category (Staff/Worker), Barcode ID, meal eligibility.

3.2 Barcode Card Integration

Each employee receives a PVC card with unique barcode.

|| || W1023 ||

Barcode scanners read employee ID (e.g., W1023).

3.3 Meal Scanning

Process: scan → identify employee → check eligibility → record → confirmation.

✅ Breakfast Approved ⛔ Breakfast Already Taken

3.4 Meal Limit Rules

MealLimit per day
Breakfast1
Lunch1

System prevents duplicate scanning.

3.5 Payroll Deduction

Example costs: Breakfast ₹15, Lunch ₹25.
Monthly: (breakfast count × ₹15) + (lunch count × ₹25) = total deduction. Export payroll report.

3.6 Reporting System

Admin dashboard: meals consumed today, worker participation, statistics, monthly summaries. Export Excel/CSV/PDF.

3.7 Offline Operation

Scan system uses local database (SQLite), records instantly, syncs to cloud when internet available. Zero interruption.

⚡ Supports low‑internet areas, fast scanning, no service downtime.
